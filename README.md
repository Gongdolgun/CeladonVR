### [프로젝트 내용]
강진 고려청자 박물관에서 전시중인 Oculus Quest 2를 활용한 청자 만들기 시뮬레이터입니다.

### [프로젝트 투입 인원]
개발자 1, 디자이너 1

### [나의 역할]
- Oculus Quest 2 연동 및 기능 테스트
- 레벨 디자인 및 기능 구현
- 버그 수정 및 현장 설치

### [주요 코드]
Motion Controller의 Trigger 액션과 Grab액션을 기반으로 전체적인 프로젝트의 틀을 잡았으며 각종 기능의 세부적인 부분을 구현하였습니다.
